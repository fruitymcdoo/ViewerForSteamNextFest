# Viewer for Steam Next Fest (June 2026)

Simple viewer for all the demos available during Steam Next Fest June 2026

Includes `viewer.html` and `nextfest_demos_2026-06-19.json`

- **viewer.html** is the viewer itself, double click or load in your browser
- **nextfest_demos_2026-06-19.json** is the steam demo data for all 4407 games from this next fest, select this file after loading the viewer

## To Use:

1. Download the repo by clicking the green `<>` Code button, then in that menu click Download ZIP
2. Extract the zip into its own folder on your computer
3. Open viewer.html by double clicking it
4. In the opened Viewer page, click Choose JSON File
5. Navigate to the location you downloaded this and select nextfest_demos_2026-06-19.json
6. The demos will load and you can browse them!
